## Description

## This p5.js project uses multiple game states and files to tell a fictional, interactive story. The user is prompted to click through the buttons, leading them to a different page. Once the user is presented with 3 separate paths, it is their job to determine the correct one to take to win the game. 2 of them lead to destruction, but 1 leads to victory.

## Setup and Interaction Instructions

- Click through the screens using the buttons provided.
- You are immersed in a story, and your job is to select the correct path to take.
